To begin coding my style guide, I opened my prefered code editor Microsoft's Visual Studio Code. Once open, I added the HTML boilerplate - being sure to link my external .css stylesheet within the head tag. From there, I created an h1 tag to let users know my style guide's name. The img src tag allowed me to share my logo by copying and pasting the url into the href portion within quotations. I separated the different sections of my style guide using the hr tag to create a horizantal line between different pages. To display my color palette, I used the div tag, and defined a width: and height: of 100px, a display: attribute set to inline-block; and a background-color: of my choosing. The paisley background was done by adding the url or local file path of the image within curly brackets in the body of my css stylesheet. Some common styling seen across my webpage is accomplished by creating a heading class which defines my font selection of Georgia, and Avenir for the body. The Avenir font can be seen beneath the colored divs as the name and hex number labels of the respective colors. Horizantal lines, define by the hr tag have a class that defines their display style as dashed, alternating between coral and teal, consistent with the style guide's color palette. At the bottom of that page, I have another link which allows viewers to download my style guide as a PDF.
